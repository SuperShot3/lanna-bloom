# Schema and SEO Implementation Notes

For each article, use:

- `BlogPosting` or `Article`
- `BreadcrumbList`
- Existing site `Organization` data
- Visible FAQ section
- `FAQPage` only if it matches the site's current schema strategy

Do not put `Product` schema on editorial articles unless a real purchasable product is visibly presented with accurate offer information.

Recommended Open Graph image: 1200 × 630.
Recommended canonical URL: the final production URL.
Do not change the Monday canonical URL.

Validate:
- Metadata length
- One H1
- Heading hierarchy
- Canonical
- OG title/description/image
- Structured data
- Internal links
- Image width/height
- Alt text
