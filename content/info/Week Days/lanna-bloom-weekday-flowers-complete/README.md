## Important implementation rule

Lanna Bloom already has a Monday flowers article built around yellow flowers. Do not blindly create a duplicate Monday page. Locate the existing Monday article and:

1. Preserve its existing URL and main content.
2. Add the pillar-page link, previous/next navigation, metadata and schema only where missing.
3. Use `01-monday-flowers.md` as editorial guidance or replacement content only if the current article is incomplete.
4. Create the pillar page and Tuesday–Sunday articles as new pages.
5. Replace all `[LINK ...]` placeholders with real internal URLs.
6. Use the supplied stock-image links only as sourcing references. Download, optimize and self-host selected images rather than hotlinking them.
7. Convert images to WebP or AVIF, keep the hero image around 1200×630, and add the supplied alt text.
8. Confirm same-day delivery wording against current operations before publishing.
