การสั่งซื้อกับ Lanna Bloom เริ่มต้นที่เว็บไซต์ของเรา หลังกด **ชำระเงินตอนนี้** คุณจะไปชำระที่หน้า **Stripe Checkout** (หน้าชำระเงินของ Stripe) Stripe อาจมีตัวเลือก**มากกว่าบัตร** — เช่น บัตรเครดิต/เดบิตหลัก (**Visa**, Mastercard ฯลฯ) **พร้อมเพย์** ในประเทศไทย และ **Apple Pay** / **Google Pay** เมื่อ Stripe รองรับ — ขึ้นกับสิ่งที่ร้านเปิดใช้กับ Stripe สกุลเงิน (**THB**) และอุปกรณ์ของคุณ **เงินสดใช้ชำระผ่าน Stripe หรือบนเว็บไม่ได้** หากจ่ายด้วยบัตร กรอกข้อมูลบัตรบนหน้าของ Stripe เท่านั้น เราไม่เก็บหมายเลขบัตรเต็มบนเว็บไซต์ของเรา

หลังชำระเงินแล้ว กรุณา[ติดต่อเราทาง LINE หรือ WhatsApp](/th/contact) แล้วส่งหมายเลขออเดอร์หรือลิงก์ออเดอร์มา เราจะยืนยันความพร้อมของสินค้าและรายละเอียดการจัดส่งให้ที่นั่น พร้อมเก็บข้อมูลไว้ในที่เดียว

การส่งหมายเลขหรือลิงก์ออเดอร์ช่วยให้เราหาออเดอร์ทันทีและประสานงานได้ราบรื่น — โดยเฉพาะเมื่อต้องขอการยืนยันด่วนจากคุณ หรือคนขับต้องการความช่วยเหลือเรื่องรายละเอียดการจัดส่ง

สำหรับ**คำขอแบบกำหนดเอง**และงานพิเศษ คุณสามารถเริ่มต้นในแชทได้ สำหรับช่อดอกไม้มาตรฐาน แนะนำให้เริ่มจากการชำระเงินผ่านเว็บไซต์

## ภาพรวม

1. เลือกช่อดอกไม้บน[เว็บไซต์](/th/catalog)
2. ใส่ตะกร้าและกรอกข้อมูลการสั่งซื้อ (ชื่อผู้รับ ที่อยู่ วัน/เวลา หมายเหตุ การ์ดข้อความ)
3. กด **ชำระเงินตอนนี้** → เปิดหน้า **Stripe Checkout** — เลือกช่องทางที่ Stripe แสดง (บัตร พร้อมเพย์ วอลเล็ต ฯลฯ ตามที่มี)
4. หลังชำระเงินสำเร็จ เก็บหมายเลขออเดอร์ + ลิงก์ออเดอร์จากหน้ายืนยันออเดอร์
5. [ติดต่อเราในแชท](/th/contact) เพื่อยืนยันความพร้อมและรายละเอียดการจัดส่งสุดท้าย
6. จัดส่งในช่วงเวลาทำการ 08:00–20:00

## ขั้นตอนทีละขั้น

### 1) สั่งซื้อและชำระเงินบนเว็บไซต์

เลือก[ช่อดอกไม้](/th/catalog) ใส่ตะกร้า แล้วกรอกข้อมูลดังนี้:

- ชื่อผู้รับ (และเบอร์โทรถ้ามี)
- ที่อยู่จัดส่ง (บ้าน / คอนโด / โรงแรม / โรงพยาบาล)
- วันจัดส่งและช่วงเวลาที่ต้องการ
- หมายเหตุสำหรับคนขับ (ตึก ชั้น ห้อง จุดสังเกต ที่จอดรถ)
- ข้อความการ์ด (ถ้าต้องการ)

เมื่อกรอกครบแล้ว กด **ชำระเงินตอนนี้** ระบบจะพาไปหน้า **Stripe Checkout** เลือกวิธีที่ Stripe เปิดให้ในหน้านั้น หลังชำระเงินสำเร็จ คุณจะได้หมายเลขออเดอร์และลิงก์ออเดอร์บนหน้าออเดอร์ของคุณ

### 2) ติดต่อเราเพื่อยืนยัน

ถัดไป ส่งข้อความมาที่:

- [ติดต่อ LINE](/th/contact)
- [ติดต่อ WhatsApp](/th/contact)

กรุณาส่งลิงก์ออเดอร์หรือหมายเลขออเดอร์มาก่อน

วิธีนี้ช่วยให้เราหาออเดอร์ได้ทันทีและยืนยันทุกอย่างได้โดยไม่สับสน

#### ส่งอะไรในแชท (คัดลอกเช็คลิสต์)

**จำเป็น**

- ส่งสวัสดี (ถึง Lanna Bloom ใน LINE หรือ WhatsApp)
- ส่งลิงก์ออเดอร์หรือหมายเลขออเดอร์

**แนะนำอย่างยิ่ง**

- หมายเหตุสำหรับคนขับ (จุดสังเกต ที่จอดรถ รหัสประตู กฎล็อบบี้)
- เบอร์โทรผู้รับ (ถ้ามี)

ทำไมถึงสำคัญ: ถ้าคนขับหาที่อยู่ไม่เจอหรือเข้าไม่ถึง รายละเอียดเหล่านี้ช่วยให้เราประสานงานได้เร็วและหลีกเลี่ยงความล่าช้าในการจัดส่ง

สำหรับ[นโยบายคืนเงิน ยกเลิก และเปลี่ยนสินค้า](/th/refund-replacement) อ่านได้ที่นี่

## จัดส่งวันเดียวและจุดตัด 18:00

เราจัดส่งในช่วงเวลาทำการ 08:00–20:00 จัดส่งวันเดียวได้เมื่อสั่งก่อน (ขึ้นอยู่กับความพร้อมของช่อดอกไม้และกำลังการจัดส่ง)

ออเดอร์ที่สั่งหลัง 18:00 อาจจัดส่งวันถัดไป บางครั้งอาจจัดส่งวันเดียวได้ แต่ขึ้นอยู่กับสต็อกและตารางจัดส่ง

ถ้าด่วน ให้ดำเนินการชำระเงินในตะกร้าให้เสร็จ แล้ว[ส่งข้อความหาเรา](/th/contact)ทันทีพร้อมหมายเลขหรือลิงก์ออเดอร์

## การชำระเงิน

วิธีชำระขึ้นอยู่กับว่าคุณใช้ **Stripe Checkout** บนเว็บไซต์ หรือชำระตามที่ประสานใน **แชท**

| | **Stripe Checkout** (กด **ชำระเงินตอนนี้** ในตะกร้า) | **LINE / WhatsApp** (เมื่อประสานกับเรา) |
|--|--------------------------------------------------------|------------------------------------------|
| **คืออะไร** | หน้าชำระเงินของ Stripe — ใช้กับช่อมาตรฐานที่สั่งบนเว็บ | เราส่ง **QR พร้อมเพย์** หรือ**ข้อมูลโอนธนาคารไทย** ในแชทหลังคุยออเดอร์แล้ว |
| **เงินสด** | **ไม่ได้** — Stripe ไม่รับเงินสดออนไลน์ | **ไม่รับ** สำหรับการชำระตามเส้นทางนี้ — ใช้พร้อมเพย์/โอนตามที่เราแจ้ง |
| **บัตร (Visa, Mastercard ฯลฯ)** | **ได้** เมื่อ Stripe แสดงในหน้านั้น | — |
| **พร้อมเพย์ไทย** | **ได้** เมื่อ Stripe แสดง **PromptPay** ในหน้าชำระ (รองรับไทย สกุล THB เมื่อเปิดใช้กับบัญชีเรา) | **ได้** เมื่อเราส่ง QR / วิธีชำระในแชท |
| **Apple Pay / Google Pay** | **ได้บางกรณี** ถ้า Stripe แสดงให้กับเบราว์เซอร์หรือเครื่องของคุณ | — |
| **โอนธนาคารไทย (นอกระบบบัญชีที่ส่งในแชท)** | เฉพาะถ้ามีวิธีโอน/ธนาคารที่ Stripe รองรับในหน้านั้น | **ได้** เมื่อเราส่งเลขบัญชีในแชท |

**เพิ่มเติม**

- **Stripe** เป็นคนเลือกว่าจะแสดงช่องทางใดบ้าง ตามกฎของการชำระ **THB** สถานการณ์ของคุณ และสิ่งที่เราเปิดใช้กับ Stripe คุณอาจเห็น**แค่บัตร** หรือ**บัตรร่วมกับพร้อมเพย์และวอลเล็ต** — เลือกตามที่ Stripe แสดง
- **เงินสด** ใช้ชำระบน Stripe หรือเช็คเอาต์ออนไลน์ของเราไม่ได้

**ออเดอร์พิเศษหรือกรณีพิเศษ**

- สำหรับ[ออเดอร์แบบกำหนดเอง](/th/custom-order) หรือหากชำระบนเว็บไม่สะดวก [ส่งข้อความหาเรา](/th/contact) เพื่อปรึกษา (เช่น พร้อมเพย์หรือโอนในแชท)

- แม่เหียะ

## คำถามที่พบบ่อย

### 1) สั่งซื้อแล้ว — ควรทำอะไรต่อ?

[ส่งข้อความมาที่ LINE หรือ WhatsApp](/th/contact) แล้วส่งลิงก์หรือหมายเลขออเดอร์มา เราจะยืนยันความพร้อมและรายละเอียดการจัดส่ง หากคุณเพิ่งชำระผ่าน Stripe การชำระเงินกำลังดำเนินการหรือสำเร็จแล้ว สถานะจะแสดงบนหน้าออเดอร์ของคุณ

### 2) แก้ไขรายละเอียดการจัดส่งหลังสั่งได้ไหม?

ได้บ่อยครั้ง ถ้าเรายังไม่ได้เริ่มเตรียมหรือส่งคนขับ ส่งข้อความมาพร้อมหมายเลขหรือลิงก์ออเดอร์โดยเร็วที่สุด

### 3) จัดส่งไปโรงแรม คอนโด หรือโรงพยาบาลได้ไหม?

ได้ กรุณาระบุห้องหรือหอผู้ป่วยถ้ามี พร้อมกฎการเข้าออก (ล็อบบี้ รีเซ็ปชัน รหัสประตู) โรงพยาบาลอาจมีข้อจำกัดในการจัดส่ง ดังนั้นการยืนยันในแชทจึงสำคัญ

### 4) เลือกเวลาจัดส่งได้แน่นอนไหม?

เลือกช่วงเวลาที่ต้องการได้ เวลาที่แน่นอนขึ้นอยู่กับการจราจรและเส้นทางจัดส่ง แต่เราจะจัดส่งตามช่วงเวลาที่คุณให้มากที่สุด

### 5) เพิ่มการ์ดข้อความได้ไหม?

ได้ เพิ่มตอนชำระเงินหรือส่งข้อความมาในแชท เขียนให้ชัดเจนและกระชับสำหรับการ์ด

### 6) ถ้าคนขับหาที่อยู่ไม่เจอล่ะ?

นี่คือเหตุผลที่เราขอหมายเหตุสำหรับคนขับและเบอร์โทรเมื่อเป็นไปได้ เราจะประสานงานในแชทและช่วยคนขับไปถึงที่ถูกต้อง

### 7) ต้องการหลักฐานการชำระเงินอะไรบ้าง?

หากชำระครบผ่าน **Stripe Checkout** หน้าออเดอร์จะอัปเดตหลัง Stripe ยืนยัน — โดยทั่วไป**ไม่ต้อง**ส่งสลิปโอน **พร้อมเพย์หรือบัตร** ที่ชำระ**บนหน้าของ Stripe** Stripe ยืนยันให้อัตโนมัติ ถ้าเราให้คุณชำระ**เฉพาะในแชท** (โอนด้วยตนเอง) ให้ส่งสลิปหรือภาพหน้าจอในแชท หากมีข้อผิดพลาด [ติดต่อเรา](/th/contact)พร้อมหมายเลขหรือลิงก์ออเดอร์

### 8) การเปลี่ยนแปลง ยกเลิก คืนเงิน

ถ้ามีอะไรเปลี่ยน ส่งข้อความมาที่เราทันทีพร้อมหมายเลขหรือลิงก์ออเดอร์ รายละเอียดนโยบายอยู่ที่นี่: [นโยบายคืนเงิน / เปลี่ยนสินค้า / ยกเลิก](/th/refund-replacement)
