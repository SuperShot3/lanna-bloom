# Lanna Bloom Social Reel Design

## Visual Identity

Soft luxury floral brand direction built for vertical social video:
- Blush pink accents over ivory gradients
- Muted rose-gold highlights for premium finish
- Warm brown typography for editorial elegance
- No photos; all decorative blooms are CSS gradients and organic rings

## Design Tokens

```css
:root {
  --bg: #f6eeea;
  --ink: #5a3d35;
  --accent: #d9a8b8;
  --muted: #9b6f63;
  --accent-dim: #c79084;
  --ivory: #fff9f3;
  --rose-gold: #cba18b;
  --warm-brown: #5a3d35;
  --font-display: "Plus Jakarta Sans", sans-serif;
  --font-data: "IBM Plex Mono", monospace;
}
```

## Typography

- Display: Plus Jakarta Sans at strong weight contrast (300/800)
- Data labels: IBM Plex Mono for precision and contrast
- Headline minimum is above 60px
- Supporting body text remains above 20px
